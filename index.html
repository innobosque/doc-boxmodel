Modelo Caja
content padding boder margin

Definición

El modo de representación de un elemento dentro de un documento web es el llamado modelo caja. Éste se ve como un rectángulo junto con sus propiedades

Como veremos a continuación en realidad un elemento está formado por varias cajas solapadas

Propiedades

margin
Es una área vacía existente entre el elemento considerado y el resto de componentes de un documento (se considera fuera del documento)
border
Es una extensión de la zona de relleno (padding). Propiedades de border:
  • border-style
  • border-width
  • border-color
padding
Es una extensión del contenido del elemento. Crea una zona vacía interna al mismo.
contenido
Es un espacio definido a través de las propiedades width y height

Tamaño de la caja

Tamaño Caja = Tamaño Contenido + padding + border

display

La propiedad display nos permite modificar el estilo de caja aplicado a un elemento dado. Entre sus valores destacamos:

display: block
Transforma un elemento HTML en bloque
display: inline
Transforma un elemento HTML en elemento en línea. Características:
  • Son apilables
  • No tienen ni margin-top ni margin-bottom
  • No respetan ni width ni height. Estas medidas dependerán del tamaño en píxels de su contenido
  • Elementos inline
    a, span, label, strong, br, input, textarea, abbr, b, basefont, bdo, big, cite, code, dfn, em, i, kbd, q, s, samp, select, small, strike, sub, sup, u, var
display: inline-block
Transforma un elemento HTML en línea y bloque simultáneamente: se comporta como un elemento en bloque pero con respecto a los elementos que la rodean es una caja en línea
